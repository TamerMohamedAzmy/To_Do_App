📝 Todo App

A simple and responsive Todo application built with React.

🚀 Features

- ➕ Add tasks
- ✏️ Update tasks
- 🗑️ Delete tasks
- ✅ Toggle completed
- 🔍 Search tasks
- 🎯 Filter (All / Active / Completed)
- 🌙 Dark Mode
- 💾 LocalStorage support

🛠️ Tech Stack

- React
- TypeScript
- CSS

▶️ Run Locally

npm install
npm start
